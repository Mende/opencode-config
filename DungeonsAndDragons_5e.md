# Dungeons & Dragons 5e Reference Guide

## Character Creation Steps

### 1. Determine Ability Scores
Choose one method:
- **Standard Array**: 15, 14, 13, 12, 10, 8
- **Point Buy**: 27 points to spend (1:1 ratio, scores above 13 cost 2 points per point)
- **Rolling**: 4d6 drop lowest, six times

### 2. Choose Race
- Select from standard races or custom options
- Apply racial ability score increases
- Note racial traits, features, and proficiencies

### 3. Choose Class
- Select class based on preferred playstyle
- Gain class features at 1st level
- Determine hit points (class hit die + Constitution modifier)
- Note saving throw proficiencies

### 4. Choose Background
- Select background that fits character concept
- Gain skill proficiencies from background
- Gain tool proficiencies and languages
- Receive starting equipment

### 5. Calculate Ability Modifiers
- **Modifier = (Score - 10) รท 2** (round down)
- Example: Score 14 = +2 modifier, Score 15 = +2 modifier

### 6. Determine Proficiency Bonus
- **Level 1-4**: +2
- **Level 5-8**: +3
- **Level 9-12**: +4
- **Level 13-16**: +5
- **Level 17-20**: +6

### 7. Calculate Skills
- **Proficient Skills**: Ability modifier + proficiency bonus
- **Expertise**: Ability modifier + (proficiency bonus ร 2)
- **Non-proficient**: Ability modifier only

**Strength**: Athletics

**Intelligence**: Arcana, History, Investigation, Nature, Religion

### Skill Check Formula
d20 + Ability Modifier + Proficiency Bonus (if proficient) + Other Bonuses

### Advantage/Disadvantage
- **Advantage**: Roll 2d20, take higher
- **Disadvantage**: Roll 2d20, take lower
- Cancel each other out if both apply

### Combat Flow

#### 1. Initiative
- Roll d20 + Dexterity modifier
- Higher acts first
- Ties: roll off or go by Dexterity scores

#### 2. Turn Structure
Each turn consists of:
- **Bonus Action**: One bonus action (if available)
- **Action**: One action (Attack, Dash, Dodge, Hide, etc.)
- **Movement**: Up to speed (can split before/after action)
- **Reaction**: One reaction (if triggered)

#### 3. Attack Roll
d20 + Strength/Dexterity modifier + Proficiency Bonus
- Must meet or beat target's Armor Class (AC)
- Natural 20 = critical hit (double damage dice)
- Natural 1 = automatic miss

#### 4. Damage Formula
Weapon/Spell Damage + Strength/Dexterity modifier (if applicable)

#### 5. Saving Throws
- **Fortitude**: Strength/Constitution saves
- **Reflex**: Dexterity saves  
- **Will**: Intelligence/Wisdom/Charisma saves
- DC = 8 + Proficiency Bonus + Ability Modifier

### Conditions
- **Prone**: Speed 0, disadvantage on attack rolls, advantage on melee attacks against prone creatures
- **Grappled**: Speed 0, can't benefit from bonuses to speed
- **Restrained**: Speed 0, attack rolls have disadvantage, attack rolls against have advantage
- **Stunned**: Can't take actions/reactions, automatically fails Strength/DEX saves
- **Unconscious**: Incapacitated, can't move/speak, unaware, drops items, falls prone

## Quick Combat Reference

### Action Types
- **Attack**: Make one or more weapon attacks
- **Cast a Spell**: Cast a spell with casting time of 1 action
- **Dash**: Double your movement
- **Disengage**: Move without provoking opportunity attacks
- **Dodge**: All attacks against you have disadvantage, your attack rolls have advantage
- **Help**: Give advantage to ally's ability check or attack
- **Hide**: Make Dexterity (Stealth) check
- **Ready**: Prepare action to trigger later
- **Search**: Make Wisdom (Perception) or Intelligence (Investigation) check
- **Use Object**: Interact with object

### Bonus Actions
- Two-weapon fighting
- Certain spells (healing word, misty step)
- Class features (Rogue's Cunning Action, Barbarian's Rage)

### Reactions
- Opportunity attacks
- Shield spell
- Counterspell
- Class features (Battlemaster's Parry, Monk's Deflect Missiles)
